# Student Management System ☕

## Project Description

This is a console-based Java application that allows users to manage student records.

## Features

* Add Student
* View All Students
* Search Student by ID
* Update Student
* Delete Student
* Exit Program

## Concepts Used

* Java OOP
* ArrayList
* Loops
* Methods
* Encapsulation
* Exception Handling

## Learning Outcome

After completing this project, you will understand how multiple Java concepts work together in a real-world application.
